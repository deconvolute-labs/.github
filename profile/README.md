# Deconvolute Labs

Building infrastructure for production AI agent systems.
We're focused on the gap between prototype and production: helping engineering teams build agents that are reliable, verifiable, and cost-efficient in the real world.

## Projects

- [deconvolute](https://github.com/deconvolute-labs/deconvolute) — Runtime policy enforcement for MCP tool calls using CEL-based rules.
- [legend](https://github.com/deconvolute-labs/legend) — PII pseudonymization for the full agentic loop. Intercepts at all four boundaries; restores originals in the final response.
- [pii-benchmark](https://github.com/deconvolute-labs/pii-benchmark) — Reproducible head-to-head benchmark for PII detection and pseudonymization. Currently compares Legend and Presidio across eleven entity types.
